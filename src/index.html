Definición

Sea `f` una función definida para `t ge 0`. Entonces se dice que la integral `\mathcal{L}\{f(t)\} = int_0^infty e^{-st} f(t)dt` es la transformada de Laplace de `f`, siempre que converja la integral.

Cuando la integral de la definicón converge, el resultado es una función de `s`. En la explicación general se usa una letra minúscula para denotar la función que se transforma y la letra mayúscula correspondiente para denotar su transformada, por ejemplo, `\mathcal{L}\{f(t)\}=F(s),\quadmathcal{L}\{g(t)\}=G(s),\quad mathcal{L}\{y(t)\}=Y(s).`

Las líneas de codigo que generan esas ecuaciones son:

  1. mathcal{L}\{f(t)\} = int_{0}^\infty e^{-st} f(t)dt
  2. mathcal{L}\{f(t)\} = F(s),\quad mathcal{L}\{g(t)\}=G(s),\quad mathcal{L}\{y(t)\}=Y(s)

Transformadas inversas

El problema inverso Si `F(s)` representa la transformada de Laplace de una función `f(t)`, es decir, `mathcal{L}\{f(t)\}=F(s)`, se dice entonces que `f(t)` es la transformada de Laplace inversa de `F(s)` y se escribe `mathcal{L}^{-1}\{F(s)\}`.

Transformadas de derivadas

Trasnformada de una derivada El objetivo inmediato de la transformada de Laplace es usarla para resolver ecuaciones diferenciales. Para este fin, es necesario evaluar cantidades como `mathcal{L}\{dy/dt\}` y `mathcal{L}\{\frac{d²y}{dt²}\}`. Por ejemplo, si `f'` es continua para `t ge 0`, entonces mediante la integración por partes se obtiene \[\begin{align*} \mathcal{L}\{f'(t)\} &=\int_0^\infty e^{-st}f'(t) dt=e^{-st}\big|_0^\infty + s\int_0^\infty e^{-st}f(t)dt\\ &=-f(0)+s\mathcal{L}\{f(t)\}\\ &=sF(s)-f(0) \end{align*}\] o bien `\quad mathcal{L}\{f'(t)\}=sF(s)-f(0)`.

Aquí se ha puesto que `e^(-st) f(t) to 0` cuando `t rarr infty`. De manera similar, con la ayuda de la ecuación (6) \[\begin{align*} \mathcal{L}\{f''(t)\} &=\int_0^\infty e^{-st}f''(t)dt=e^{-st}f'(t) \big|_0^\infty + s \int_0^\infty e^{-st}f'(t)dt\\ &= -f'(0)+s \mathcal{L}\{f'(t)\}\\ &=s[sF(s)-f(0)]-f'(0) \end{align*}\] o bien `\quad mathcal{L}\{f''(t)\}=s^2F(s)-sf(0)-f'(0)`.

De igual manera se puede demostrar que `mathcal{L}\{f'''(t)\}=s^3F(s)-s^2f(0)-sf'(0)-f''(0)`.

La naturaleza recursiva de la transformada de Laplace de las derivadas de una función f es evidente de los resultados en las ecuaciones anteriores.

Derivada de una transformada

Multiplicación de una función por `t^n` La transformada de Laplace del producto de una función `f(t)` con `t` se puede encontrar mediante diferenciación de la transformada de Laplace de `f(t)`. Para motivar este resultado, se supone que `F(s)=mathcal{L}\{f(t)\}` existe y que es posible intercambiar el orden de diferenciación e integración. Entonces `\frac{d}{ds}F(s)=\frac{d}{ds} \int_0^\infty e^{-st}f(t)dt=\int_0^\infty \frac{\partial}{\partial s}[e^{-st}f(t)dt]=-\int_0^\infty e^{-st}tf(t)dt=-\mathcal{L}\{tf(t)\}`; es decir, `\quad \mathcal{L}\{tf(t)\}=-\frac{d}{ds}\mathcal{L}\{f(t)\}`.

Se puede usar el último resultado para hallar la transformada de Laplace de `t^nf(t)`: \[\mathcal{L}\{t^2 f(t)\}=\mathcal{L}\{t·ft(t)\}=-\frac{d}{ds}\mathcal{L}\{tf(t)\}=-\frac{d}{ds} (-\frac{d}{ds}\mathcal{L}\{f(t)\}) = \frac{d^2}{ds^2}\mathcal{L}\{f(t)\}\]

Trasformadas de integrales

Convolución Si las funciones `f` y `g` son continuas por partes en `[0, infty)`, entonces un producto especial, denotado por `f**g`, se define mediante la integral `f*g=int_0^t f(\tau)g(t-\tau)d\tau` y se llama convolución de `f` y `g`. La convolución de `f**g` es una función de `t`. Por ejemplo, `e^t*\sin t=int_0^t e^{\tau}\sin(t-\tau)d\tau=\frac{1}{2}(-\sin t-\cos t +e^t).`

Esto se dejó como un ejercicio para mostrar que `int_0^t f(\tau)g(t-\tau)d\tau=int_0^t f(t-\tau)g(\tau)d\tau;` es decir, `\quad f**g=g**f`. Esto significa que la convolución de dos funciones es conmutativa.

No es cierto que la integral de un producto de funciones sea el producto de integrales. Sin embargo, es cierto que la transformada de Laplace del producto especial, es el producto de la transformada de Laplace de `f` y `g`. Esto significa qe es posible determinar la transformada de Laplace de la convolución de dos funciones sin evaluar en realidad la integral como se hizo.

El código que genera las última ecuaciones de esta sección es:

  1. `f*g=int_0^t f(\tau)g(t-\tau)d\tau`
  2. `e^t*\sin t=int_0^t e^{\tau}\sin(t-\tau)d\tau=\frac{1}{2}(-\sin t-\cos t +e^t)`
  3. `int_0^t f(\tau)g(t-\tau)d\tau=int_0^t f(t-\tau)g(\tau)d\tau.`

Transformada de una función periódica

Función periódica Si una función periódica tiene un periodo `T`, `T ge 0`, entonces `f(t+T)=f(t)`. El siguiente teorema muestra quela transformada de Laplace de una función periódica se obtiene mediante integración sobre un periodo.

Si `f(t)` es continua por partes en `[0,infty)`, de orden exponencial y periódica con periodo `T`, entonces `mathcal{L}\{f(t)\}=\frac{1}{1-e^{-sT}}int_0^T e^{-st}f(t)dt.`

El codigo que genera la fórmula en esta sección es:

  1. `mathcal{L}\{f(t)\}=\frac{1}{1-e^{-sT}}int_0^T e^{-st}f(t)dt`
